### 1. Clone the repository

```
git clone <REPOSITORY_URL>
cd march-madness-app
```

### 2. Navigate to the frontend

```
cd frontend
```

### 3. Install dependencies

```
npm install
```

### 4. Start the development server

```
npm run dev
```
